# 0xchou00 Web

React website for 0xchou00 — Lightweight Security Detection Tool.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deploy

- Vercel: import the `web/` directory as the project root
- Netlify: use `npm run build` with publish directory `dist`

Both SPA rewrite configs are included:

- `vercel.json`
- `netlify.toml`
